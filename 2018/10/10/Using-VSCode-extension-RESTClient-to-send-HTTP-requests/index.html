最近在参加中国科学技术大学第五届信息安全大赛时，经高人指点，本渣找到了一款比Postman要方便（个人认为）的发送HTTP请求的工具，就是VSCode扩展REST Client，特此写博客记述一下。

安装

非常简单，在VSCode按F1，输入ext install后搜索rest-client即可。

简单GET

然后使用快捷键Ctrl+Alt+R即可运行，结果将会显示在右栏。

后记
